# Star Battle（星际大战）

![](https://ws1.sinaimg.cn/large/006mS5wEgy1fx9ouu9sncj3063049gll.jpg)

Star Battle 是类型雷电战机游戏的一款游戏，游戏操作说明：

1. 按 `W` `A` `S` `D` 或者 右下方的控制区来操作战机的移动。
2. 按 `Space` 发射。
3. 按 `P` 切换 暂停 与 继续。
4. 按 `M` 切换 静音 与 扬声。
